+disputils
+=========
+
+disputils is a set of utilities to manage multiple monitors with any window
+manager that works with wmutils.
+
+This is a C rewrite of @ix's [disputils](https://arcetera.moe/git/disputils/log.html).
+
+Credits
+------
+
+Originally based off @laserswald's [multihead pull request](https://github.com/wmutils/opt/pull/7) to [wmutils/opt](https://github.com/wmutils/opt).
+
+Especially thanks to z3bra and dcat for work on wmutils itself.
