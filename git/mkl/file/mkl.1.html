.Dd $Mdocdate: August 4 2015 $
.Dt MKL 1
.Os mkl
.Sh NAME
.Nm mkl
.Nd Draws a thick line based on input level
.Sh SYNOPSIS
.Nm mkb
.Ar [perc]
.Sh DESCRIPTION
The
.Nm
utility draws a thick line based on the input level given as argument
or from stdin using Unicode Block Elements.
.Bl -tag -width Ds
.It perc
A percentage value to be represented as a progress bar. If this argument isn't provided,
.Nm
will attempt to read the value from stdin.
.El
.Sh ENVIRONMENT
.Bl -tag -width Ds
.It Ev SIZE
The number of virtual blocks used to represent the line. A character drawn on
the terminal can be divided in 8 virtual blocks. (default: 80 virtual blocks, 10 terminal chars)
.It Ev ORIENT
A single char representing the style of the line. '|' for a vertical line, '-' for a horizontal
line. (default: '|')
.It Ev SPACE
The space char used to fill blank blocks in horizontal orientation. (default: ' ')
.El
.Sh EXAMPLES
.Dl $ mkl 75
