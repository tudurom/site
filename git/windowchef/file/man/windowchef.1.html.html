<p class="man-name">
  <code>windowchef</code> - <span class="man-whatis">A static window cooker</span>
</p>

<h2 id="SYNOPSIS">SYNOPSIS</h2>

<p><code>windowchef</code> [-h]</p>

<h2 id="DESCRIPTION">DESCRIPTION</h2>

<p><code>windowchef</code> is a stacking window manager that doesn't handle keyboard or
pointer inputs. It is controlled and configured by <code>waitron</code>.</p>

<p>At startup,
<code>windowchef</code> runs a script located at <code>$XDG_CONFIG_HOME/windowchef/windowchefrc</code>
(<code>$XDG_CONFIG_HOME</code> is usually <code>~/.config</code>). The path of the config file can be
overridden with the <code>-c</code> flag.</p>

<h2 id="OPTIONS">OPTIONS</h2>

<dl>
<dt class="flush"><code>-h</code></dt><dd><p>  Print usage and version information.</p></dd>
<dt><code>-c</code> <var>config_path</var></dt><dd><p>  Load script from <var>config_path</var> instead of
  <code>$XDG_CONFIG_HOME/windowchef/windowchefrc</code>.</p></dd>
</dl>
