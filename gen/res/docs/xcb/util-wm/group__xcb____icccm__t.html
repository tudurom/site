Parameters

Returns

Allow to get a window property, in most case you might want to use above functions to get an ICCCM property for a given window.

xcb_get_property_cookie_t  cookie,
xcb_icccm_get_text_property_reply_t *  prop,
xcb_generic_error_t **  e 
)

Fill given structure with the property value of a window.

Parameters
cThe connection to the X server.
cookieTextProperty request cookie.
propTextProperty reply which is to be filled.
eError if any.
Returns
Return 1 on success, 0 otherwise.

If the function return 0 (failure), the content of prop is unmodified and therefore the structure must not be wiped.

The parameter e supplied to this function must be NULL if xcb_icccm_get_text_property_unchecked() is used. Otherwise, it stores the error if any. prop structure members should be freed by xcb_icccm_get_text_property_reply_wipe().
